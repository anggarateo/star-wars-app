## Requirement

Node >=20.16.0

NPM >=10.8.1

## Setup

Rename .env.example to .env

Install the dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000` or `your.ip:3000`:

```bash
npm run dev
```
